# PetCare – вебзастосунок для власників домашніх тварин

## Опис проєкту
PetCare — це вебзастосунок, розроблений у межах курсової роботи, призначений для реєстрації домашніх тварин, 
управління даними власників, ветеринарних подій, сервісів та довідкової інформації.
Система орієнтована на власників тварин, ветеринарні клініки, тренерів, заводчиків та організації, що допомагають тваринам.

## Функціональні можливості
- реєстрація та авторизація користувачів;
- додавання та перегляд домашніх тварин;
- ведення ветеринарних подій (щеплення, огляди);
- перегляд новин та оголошень;
- облік кінологічних центрів та тренерів;
- каталог заводчиків;
- перелік організацій допомоги тваринам;
- адміністративна панель зі статистикою.
## Архітектура проєкту
Проєкт побудований за клієнт-серверною архітектурою з використанням REST API:
- **Клієнт** — HTML, CSS, JavaScript (веббраузер)
- **Сервер** — Node.js + Express.js
- **База даних** — MySQL
## Використані технології
- **HTML5** — структура вебсторінок  
- **CSS3** — оформлення та адаптивний дизайн  
- **JavaScript** — клієнтська логіка  
- **Node.js** — серверне середовище виконання  
- **Express.js** — створення REST API  
- **MySQL** — зберігання даних  
## Вимоги до середовища
- Node.js (v18 або новіша)
- npm
- MySQL
- Браузер Google Chrome або Microsoft Edge
**Додатково:**  
- REST API для взаємодії між фронтендом і бекендом.  
- CRUD операції для всіх сутностей.
- ## Встановлення та запуск
1. Клонувати репозиторій:
git clone https://github.com/dmytriievhlib/pet_app_project.git
cd pet_app_project/backend
2. Встановити залежності:
npm install
3. Створити базу даних MySQL та імпортувати db_init.sql.
4. Запустити сервер:
node server.js
5. Відкрити у браузері:
http://localhost:3000
## Використання

- login.html – вхід у систему

## API
- /api/pets – список тварин, додавання, видалення
- /api/events – список ветеринарних подій, додавання, видалення
- /api/trainers, /api/centers – управління кінологічними центрами та тренерами
- /api/breeders – заводчики
- /api/leisure – місця для дозвілля тварин
- /api/news – новини та оголошення
- /api/regulations – нормативно-правова база
- /api/organizations – організації допомоги тваринам
